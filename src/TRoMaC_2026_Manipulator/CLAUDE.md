## Architecture

This is a **MoveIt 2 configuration package** generated by MoveIt Setup Assistant, extended with two custom C++ nodes.

### Robot Model

- URDF source lives in the separate **`manipulator`** package (`urdf/manipulator.urdf`); this package only references it via `.setup_assistant` and the xacro files in `config/`.
- 6-DOF arm with joints: `yaw-1-joint`, `pitch-1-joint`, `pitch-2-joint`, `roll-1-joint`, `pitch-3-joint`, `roll-2-joint`
- End-effector frame: `roll-2_Link`; planning group name: `manipulator`
- IK solver: **TRAC_IK** (`config/kinematics.yaml`)
- `config/joint_limits.yaml`: velocity/acceleration limits are currently disabled (all set to false/0) ‚Äî only the URDF limits apply.

MoveIt Servo (`moveit_servo`) provides real-time Cartesian/joint jogging as an alternative path to the same `JointTrajectoryController`.

### Custom Nodes (`src/`)

**`serial_comm_node`** (`serial_comm_node.cpp` + `Serial.h`/`Serial.cpp`):
- Bridges ROS 2 to the physical MCU over UART
- **TX frame** (12 bytes): 6 √ó `int16_t` joint angles in integer degrees ‚Üí `VisionFrameTX_structTypedef`
- **RX frame** (12 bytes): arm position x/y/z, pitch, roll, button, end-byte ‚Üí `VisionFrameRX_structTypedef`
- Subscribes to `serial_send` (`std_msgs/Float64MultiArray` with 6 doubles) for on-demand sends
- Parameters: `device` (default `/dev/ttyACM0`), `baud_rate` (default 921600), `auto_send` (default true), `send_rate_hz` (default 100.0), `tx_joints` (6 doubles), `send_once_on_start`, `log_auto_send`
- A background thread reads incoming frames using `select()` with 300 ms timeout

**`servo_keyboard_input`** (`servo_keyboard_input.cpp`):
- Keyboard teleoperation for MoveIt Servo
- Calls `/servo_node/start_servo` service on startup (waits up to 15 s)
- Publishes `geometry_msgs/TwistStamped` ‚Üí `/servo_node/delta_twist_cmds` for Cartesian jog
- Publishes `control_msgs/JointJog` ‚Üí `/servo_node/delta_joint_cmds` for per-joint jog
- Opens `/dev/tty` as fallback when stdin is not a terminal (ros2 launch case); if no TTY is available it throws and exits cleanly

### Key Config Files

| File | Purpose |
|------|---------|
| `config/ros2_controllers.yaml` | ros2_control: `manipulator_controller` (JointTrajectoryController) + `joint_state_broadcaster` |
| `config/moveit_controllers.yaml` | MoveIt controller manager mapping to `manipulator_controller` |
| `config/servo_settings.yaml` | Servo: unitless commands, 0.01 s period, Butterworth smoothing, collision checking enabled |
| `config/kinematics.yaml` | TRAC_IK, 5 ms timeout |
| `config/pilz_cartesian_limits.yaml` | Pilz planner Cartesian velocity/acceleration limits |
| `config/*.xacro` | Robot description with ros2_control hardware interface |
| `config/*.srdf` | MoveIt semantic description (planning groups, named states) |
